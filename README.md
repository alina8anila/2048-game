## План роботи
1. Паращій Аліна -> *Ігрова логіка*
- Реалізація процедури ініціалізації гри: очищення поля та генерація перших двох випадкових плиток
- Написання алгоритму cтиснення: розробка логіки переміщення всіх ненульових значень до одного краю ряду без пробілів
- Написання алгоритму злиття: реалізація правил додавання однакових сусідніх чисел із захистом від подвійного злиття
- Реалізація логіки для всіх 4 сторін (Вгору, Вниз, Вліво, Вправо) через трансформацію координат або реверс рядків/стовпців
- Написання алгоритму пошуку випадкової порожньої клітинки та розміщення там «2» або «4»
- Визначення порожнього ходу через порівняння стану поля до і після натискання клавіші
- Перевірка поразки та перемоги
- Обчислення рахунку: додавання значення злитих плиток до загального балу
- Скидання рахунку та стану гри при перезапуску
+ Функція "Скасувати хід" (Undo)
+ Робота з перериванням INT 21h для запису та зчитування найкращого результату з файлу
2. Сахарова Юлія -> *Рендеринг та ввід*
- Відображення ігрового поля: рендеринг клітинок (кожна плитка - багатосимвольний прямокутник, централізоване значення, порожні клітинки відображаються як рамки без чисел), збереження меж поля та перемальовування після ходу гравця
- Атрибути кольорів, таблиця пошуку (плитки різних значень - різні кольори)
- Обробник клавіатури, що  реагує на хід гравця (зчитування клавіш через int 16h, можливість завершити/перезапустити гру клавішами "ESC" та "R")
- Відображення поточного та найкращого рахунку гравця
- Відображення базових інструкцій для керування грою 
- Вивід повідомлень "GAME OVER" та "YOU WIN" поверх ігрового поля
- Перетворення двійкових значень у десяткові рядки через ASCII для коректного відображення
- Поетапне незалежне тестування для перевірки правильного форматування та відображення
- Після зчитування нового стану гри, поле з клітинками перемальовується, а рахунки оновлюються
+ Параметризований розмір сітки через змінні (координати та розмір клітинок залежать від параметрів, вони НЕ є константами)
+ Покадрова анімація зсуву на основі таймера
